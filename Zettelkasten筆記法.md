Zettelkasten 筆記法的核心概念，跟寫 code 時的 「function 只做一件事」很類似

總結幾個 Zettlekasten 的特點：

- 原子化: 一篇只記一件事
- 獨立化: 可以單獨被看懂
- 建立連結
- 說自己的話
- 註解，像寫 code 一樣
